# RAG Question Answering with FLAN-T5

Retrieval-Augmented Generation (RAG) project developed for the Natural Language Processing course at USI.

The project focuses on question answering using the RAG-12000 dataset, combining retrieval-based context with transformer-based answer generation.

## Project Overview

The objective of this project was to explore modern NLP techniques for context-aware question answering systems.

Our work included:
- Dataset exploration and preprocessing
- Retrieval-Augmented Generation (RAG) pipelines
- Transformer-based answer generation
- Fine-tuning and evaluation of FLAN-T5 models
- Analysis of model performance and generated responses

The project was developed as part of a group assignment for the NLP course.

## Dataset

- Dataset: RAG-12000
- Source: https://huggingface.co/datasets/neural-bridge/rag-dataset-12000

The dataset contains:
- Questions
- Context passages
- Generated answers

It is designed for Retrieval-Augmented Generation tasks.
